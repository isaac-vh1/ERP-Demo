# Landscaping Enterprise Resource Planner (ERP)

Built for my business, **Acres By Isaac**, this ERP system improves operational efficiency by streamlining communication, centralizing data, and automating key workflows between the company and its clients.

⸻

# Overview

This application was designed to manage day-to-day business operations, including invoicing, scheduling, and financial tracking. By consolidating these processes into a single system, it reduces manual work and improves overall efficiency.

⸻

# Features
	•	Centralized client and job management
	•	Automated communication and workflow tracking
	•	Expense and invoicing tracking
	•	Scheduling and operational planning
	•	Scalable architecture for future feature expansion

⸻

# Tech Stack
	•	Frontend: React
	•	Backend: Python
	•	Database: MySQL
	•	Deployment: Docker, Self-hosted

⸻

# Motivation

This project was created to solve real operational challenges within a growing landscaping business, replacing manual processes with a scalable and efficient software solution.
